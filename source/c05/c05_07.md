# 5.7 记录过去的专属剪切板

![](http://image.iswbm.com/20200804124133.png)

在 Windows 上有一个剪切板神器 - `Ditto` ，它可以将你间经复制粘贴过的内容都保存下来，以便你重复使用。

当然在 Mac 上的 Alfred 也有类似的功能。

前段时间，偶然在电脑上偶然发现 PyCharm 也有类似的功能，只要你按住 `Ctrl + Shift + V` 就可以调出像下面这样的剪切板。
